# Welcome to CU Boulder's LLM Chat Interface! 
This interface is powered by [Chainlit](https://docs.chainlit.io), an open-source framework for building LLM-powered chat interfaces. For documentation on using this application, please visit [the CURC LLM Chat Interface documentation](https://curc.readthedocs.io/en/latest/open_ondemand/llm_chat_interface.html).
